Simponic's State System

Developed for Kulyukin's CS5000

State: _


  • must have some instruction quadruple with the start state "q0"
  • all computations will fail unless they end on the state "f"
  • instructions are delimited with line breaks and are in the form "< from_state > < read_symbol > < L/R/write_symbol > < to_state >"
  • comments ("//") are ignored
